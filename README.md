# Team Management App

📌 **Project Description**

It is a modern web application for team management, onboarding, and authentication. Built with Next.js, TypeScript, and Tailwind CSS, it provides a seamless experience for admins and participants to manage teams, handle join requests, and authenticate securely.

The app features a responsive UI, admin dashboard, and robust API routes for all core operations. Authentication and database management are powered by Supabase.

---

🚀 **Features**

- Team creation, joining, and management
  
- Admin dashboard and authentication
  
- Participant onboarding and login
  
- API routes for team and user operations
  
- Responsive UI with reusable components
  
- Secure authentication with Supabase

---

🛠️ **Tech Stack**

- Next.js
  
- TypeScript
  
- Tailwind CSS
  
- Supabase
  
- React

⚙️ **Installation & Setup**

1. Clone the repository:
   
   ```bash
   
   git clone https://github.com/YOUR_USERNAME/scrs-gravitas-app.git
   
   cd scrs-gravitas-app
   
3. Install dependencies:

   npm install

3. Set up environment variables:

   Copy .env.example to .env.local and fill in your Supabase credentials and other required variables.
   Start the development server:

4. Open in your browser:

   http://localhost:3000
